\# ALCOA+ Data Integrity Checker



Questo progetto analizza file CSV di registri operativi

per identificare possibili violazioni di Data Integrity

secondo i principi ALCOA+.



\## Funzioni principali

\- Controllo campi obbligatori

\- Verifica coerenza timestamp

\- Tracciabilità modifiche

\- Segnalazione anomalie



\## Output

Report CSV con anomalie e severità.
